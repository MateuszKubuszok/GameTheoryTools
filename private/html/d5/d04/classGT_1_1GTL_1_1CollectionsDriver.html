Detailed Description

template<typename Content>
class GT::GTL::CollectionsDriver< Content >

Used for handling collections via Driver.

Used by Parser to create collections of ValidableSymbols and insert elements into them. On Parser's they are used everywhere where some ValidableSymbols can be listed with , character.

Example of piece of code recognized by Parser:

identifier1, identifier2, identifier3
