ResultPtr GT::Routines::MatrixMixedEquilibriumRoutine::findResultFor ( const GamePtr  game,
const ConditionsPtr  conditions 
) const
overridevirtual

Returns mixed strategy equilibrium for Strategic Game with 2 Players when game is 0-sum type.

Routine uses linear programming to solve the problem. It's a typical 0-sum strategic game solution where primal solution is (distribution of Player 2 strategies ) * cost function value and dual solution is (distribution of Player 1 strategies ) * cost function value.
