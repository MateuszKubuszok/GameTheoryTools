virtual bool GT::Model::DataPiece::is0Sum ( ) const
pure virtual

Returns whether this Data piece is 0-sum one.

DataPiece is 0-sum if it's defined for 2 Players with 1 Payoff (first Player's one) or if sum of Payoffs is equal to 0.

Returns
true if it is 0-sum data piece
