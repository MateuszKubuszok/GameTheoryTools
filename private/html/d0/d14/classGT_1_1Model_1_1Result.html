Detailed Description

Root of all Results created by GTL Parser.

Since all classes of GT (except factories, Numbers, Identifiers and Messages) extend this one, they are guaranteed to have Result::toSting() and Result::isEqual(const Root&) methods as well as Result#==(const ResultPtr&, const ResultPtr&), Result#!=(const ResultPtr&, const ResultPtr&) and Result#<<(OutputStream&, const ResultPtr&) operators overloaded.
