35 /**
36  * @class Coordinate
37  * @brief Contains some Coordinate as well as data bound to it.
38  *
39  * They can be considered pieces from which sets of pure strategies will be built. For strategic-form games it
40  * means putting together all player-strategy pairs as to create whole vector for payoff function. For
41  * extensive-form games it means going from root to leafs, one node at a time.
42  *
43  * Example of piece of code recognized by Parser:
44  *
45  * @code
46  {
47  p1 = p1s1 :
48  { p2 = p2s1 : 10, 20 },
49  { p2 = p2s2 : 30, 40 }
50  },
51  {
52  p1 = p1s2 :
53  { p2 = p2s1 : 50, 60 },
54  { p2 = p2s2 : 70, 80 }
55  }
56  @endcode
57  *
58  * @author Mateusz Kubuszok
