Detailed Description

Root of all Results created by GTL Parser.

Since all classes of GT (except factories, Numbers, Identifiers and Messages) extend this one, they are guaranteed to have t::toSting()const and Result::isEqual(const Root&) methods as well as operator==(const ResultPtr&, const ResultPtr&), operator!=(const ResultPtr&, const ResultPtr&) and operator<<(OutputStream&, const ResultPtr&) operators overloaded.
